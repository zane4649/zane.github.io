Python冒泡法排序

发表于 2019-08-27 分类于 Python 阅读次数：

冒泡法排序是一种简单的排序算法，它的思想是，重复的遍历要排序的列表，一次比较2个元素，如果他们的排序错误就把他们的位置交换过来，直到没有需要再交换的元素，表明该列表已经排序完成。

例子：
有下面这样一个列表，利用冒泡法实现排序

li=[10,6,4,5,7,9,2,1,3,8]

代码如下：

li=[10,6,4,5,7,9,2,1,3,8]
for i in range(len(li)-1):
#列表中一共有10个数，两两比较，因此实际次数为9

    for j in range(len(li)-1-i):
    #每次冒泡确认一个最大值，因此只需要n-1冒泡

        if li[j] > li[j+1]:
        #判断前面的数是否大于后面的数

           li[j],li[j+1] = li[j+1],li[j]
           #如果大于则交换位置

print(li)
#打印排序后的列表
