10，求10项目内斐波那契数

代码如下：
def feibo(n):
if n <= 1:
    return n
else:
    return(feibo(n-1)+feibo(n-2))

num =int(input("你需要打印到第几项: "))
for i in range(num):
    print(feibo(i))
